### 프로젝트 소개
맛있는 커피, 디저트를 찾아서  
카페를 운영하는 영세자영업자 이익 증대를 위한 프로젝트  
게임 컨셉을 차용해 서비스 이용자에게 새로움을 선사하는 웹사이트 개설을 목표로 함  

기획 : 2023.06.21 ~ 2023.06.30  
개발 : 2023.07.01 ~ 2023.08.02

## 구현 기능
* 홈
* 카페 찾기 
* 길드
* 이벤트
* 상점
* 마이페이지
* 로그인/회원가입
* 관리자페이지
